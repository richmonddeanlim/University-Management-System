# Python Group Assignment

## Overview
This repository contains a Python group assignment developed as part of a university coursework project.  
The project focuses on applying Python programming concepts such as problem solving, data handling, and modular programming.

The goal of this assignment is to demonstrate collaboration in software development and the implementation of Python-based solutions to real-world problems.

## Features
- Implementation of Python-based solutions for assignment tasks
- Structured code organization
- Collaborative group development
- Practical application of programming concepts

## Technologies Used
- Python
- Git & GitHub
- Command Line Interface (CLI)

## Skills Demonstrated
- Python programming
- Logical problem solving
- Code collaboration using GitHub
- Code organization and documentation


## Installation
Clone the repository
```
git clone https://github.com/richmonddeanlim/Group-Assignment-Python.git
```
Navigate to the project directory
```
cd Group-Assignment-Python
```
Run the program
```
python main.py
```
